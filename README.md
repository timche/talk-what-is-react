# What is React?

> Slides from my Talk at HannoverJS in July 2016

[Show me the Slides!](https://timche.github.io/what-is-react)

## Running the Slides locally

```bash
$ git clone https://github.com/timche/what-is-react.git
$ cd what-is-react
$ npm install
$ npm start
```

## Running the Example

```bash
$ cd example
$ npm install
$ npm start
```
